.. _EEex_SetMask:

===================================
EEex_SetMask 
===================================

::

   EEex_SetMask(original, toSetMask)

Sets all the bit values in *original* to 1 that match the bitwise mask in the *toSetMask*

**Parameters**

* **original**
* **toSetMask**

**Return**

Returns the *original* value adjusted with the new bitsString mask

**Example**

::

   EEex_SetMask()
