===================================
EEex_UnsetMask 
===================================

::

   EEex_UnsetMask(original, toUnsetmask)

Sets all the bit values in *original* to 0 that match the bitwise mask in the *toUnsetmask*

**Parameters**

* **original**
* **toUnsetmask**

**Return**

Returns the *original* value adjusted with the new bitsString mask

**Example**

::

   EEex_UnsetMask()
