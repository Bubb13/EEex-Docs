===================================
EEex_AreBitsSet 
===================================

::

   EEex_AreBitsSet(original, bitsString)

Returns true if bits set in *bitString* match bits set in *original*. 

**Parameters**

* **original** - original bit settings to check against
* **bitsString** - the string containing bits to check

**Return**

true or false

**Example**

::

   EEex_AreBitsSet()
