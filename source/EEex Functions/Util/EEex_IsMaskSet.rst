.. _EEex_IsMaskSet:

===================================
EEex_IsMaskSet 
===================================

::

   EEex_IsMaskSet(original, isSetMask)

Determines if a mask value matches

**Parameters**

* **original** - original bit settings to check against
* **isSetMask** - the mask value to match against

**Return**

Returns true if mask value matches

**Example**

::

   EEex_IsMaskSet()
