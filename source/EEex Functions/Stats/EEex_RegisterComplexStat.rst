===================================
EEex_RegisterComplexStat 
===================================

::

   EEex_RegisterComplexStat(name, attributeTable)

Returns offset to the complex stat being registered

**Parameters**

* **name** - name of complex stat to register
* **attributeTable** - the attribute table to use

**Return**

Returns offset to the complex stat being registered

**Example**

::

   EEex_RegisterComplexStat()
