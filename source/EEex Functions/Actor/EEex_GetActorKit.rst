.. _EEex_GetActorKit:

===================================
EEex_GetActorKit 
===================================

::

   EEex_GetActorKit(actorID)

Returns the kit for the actor

**Parameters**

* **actorID** - actor id to return the kit for

**Return**

Returns a ``BYTE`` value indicating kit value

**Example**

::

   local kitid = EEex_GetActorKit(actorID)
