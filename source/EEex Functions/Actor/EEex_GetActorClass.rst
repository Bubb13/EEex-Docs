===================================
EEex_GetActorClass 
===================================

::

   EEex_GetActorClass(actorID)

Returns the actor's class 

**Parameters**

* **actorID** - actor id to return class for

**Return**

Returns a ``BYTE`` value indicating class

**Example**

::

   local targetclass = EEex_GetActorClass(actorID)
