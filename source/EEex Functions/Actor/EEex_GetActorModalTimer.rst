===================================
EEex_GetActorModalTimer 
===================================

::

   EEex_GetActorModalTimer(actorID)

Returns the modal timer value for the actor

**Parameters**

* **actorID** - the actor's id to return the modal timer for

**Return**

Returns a timer value

**Example**

::

   EEex_GetActorModalTimer()
