===================================
EEex_GetActorSpecificsScript 
===================================

::

   EEex_GetActorSpecificsScript(actorID)

Returns the actor's specifics script resref as a string, (defined at offset ``0x78`` of the actor structure).

**Parameters**

* **actorID** - the actor's id 

**Return**

Returns a string

**Example**

::

   EEex_GetActorSpecificsScript()
