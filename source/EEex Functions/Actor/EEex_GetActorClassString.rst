EEex_GetActorClassString 
===================================

::

   EEex_GetActorClassString(actorID)

Returns actor class as a string

**Parameters**

* **actorID** - actor id to return class string for

**Return**

Returns actor class as a string

**Example**

::

   local classname = EEex_GetActorClassString()
