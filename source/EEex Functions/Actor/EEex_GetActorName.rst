===================================
EEex_GetActorName 
===================================

::

   EEex_GetActorName(actorID)

Returns actor's name as a string

**Parameters**

* **actorID** - actor id to return name for

**Return**

Returns a string containing actor's name

**Example**

::

   local name = EEex_GetActorName(actorID)
