===================================
EEex_GetActorRace 
===================================

::

   EEex_GetActorRace(actorID)

Returns the actor's race

**Parameters**

* **actorID** - actor id to return race for

**Return**

Returns a ``BYTE`` value indicating race

**Example**

::

   local race = EEex_GetActorRace(actorID)
