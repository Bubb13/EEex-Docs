EEex_SetActionTarget 
===================================

::

   EEex_SetActionTarget(actionData, newTarget)

Set target of action data 

**Parameters**

* **actionData** - action data to set target for
* **newTarget** - new target to set

**Return**

Returns new target of action data

**Example**

::

   EEex_SetActionTarget()
