===================================
EEex_GetListScroll 
===================================

::
   
   EEex_GetListScroll(listName)

Get list scroll position

**Parameters**

* **listName** - list to return scroll positon for

**Return**

Returns scroll position

**Example**

::

   local scrollpos = EEex_GetListScroll(listName)
