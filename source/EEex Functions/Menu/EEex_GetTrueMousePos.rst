===================================
EEex_GetTrueMousePos 
===================================

::

   EEex_GetTrueMousePos()

Returns x and y coordinate for mouse position

**Parameters**

None

**Return**

Returns and x and y coordinate representing the mouse position

**Example**

::

   local mouseX, mouseY = EEex_GetTrueMousePos()
