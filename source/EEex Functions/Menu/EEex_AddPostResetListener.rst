===================================
EEex_AddPostResetListener 
===================================

::

   EEex_AddPostResetListener(listener)

Adds the specified listener function to be called after an F5 UI reload is executed.

**Parameters**

* **listener** - function to add

**Return**

None

**Example**

::

   EEex_AddPostResetListener()
