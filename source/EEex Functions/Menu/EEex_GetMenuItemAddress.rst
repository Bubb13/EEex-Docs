EEex_GetMenuItemAddress 
===================================

::

   EEex_GetMenuItemAddress(menuItemName)

Return an address of an menu item

**Parameters**

* **menuItemName** - menu item name to return address for

**Return**

Returns menu item address

**Example**

::

   EEex_GetMenuItemAddress()
