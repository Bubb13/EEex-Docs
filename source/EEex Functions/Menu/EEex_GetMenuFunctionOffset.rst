===================================
EEex_GetMenuFunctionOffset 
===================================

::

   EEex_GetMenuFunctionOffset(typeName)

Returns an array of menu offsets related to ``onopen``, ``onclose`` and ``enabled``

**Parameters**

* **typeName**

**Return**

Returns an array of menu offsets

**Example**

::

   EEex_GetMenuFunctionOffset()
