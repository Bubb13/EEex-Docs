===================================
EEex_LearnInnateSpell 
===================================

::

   EEex_LearnInnateSpell(actorID, resref)

Adds a spell to known innate spells for an actor

**Parameters**

* **actorID** - id of actor that is learning spell
* **resref** - resource reference of spell to learn 

**Return**

None

**Example**

::

   EEex_LearnInnateSpell()
