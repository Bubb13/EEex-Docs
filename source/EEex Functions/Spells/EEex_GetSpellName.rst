.. _EEex_GetSpellName:

===================================
EEex_GetSpellName 
===================================

::

   EEex_GetSpellName(resrefLocation)

Returns a string containing spell name

**Parameters**

* **resrefLocation** - resource reference of spell to return name for

**Return**

Returns a string

**Example**

::

   local name = EEex_GetSpellName("SPWI304")
