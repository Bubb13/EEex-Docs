===================================
EEex_MemorizeWizardSpell 
===================================

::

   EEex_MemorizeWizardSpell(actorID, level, resref)

Memorize a wizard spell for an actor

**Parameters**

* **actorID** - id of actor that is memorizing spell
* **level** - spell level
* **resref** - resource reference of spell to memorize

**Return**

None

**Example**

::

   EEex_MemorizeWizardSpell()
