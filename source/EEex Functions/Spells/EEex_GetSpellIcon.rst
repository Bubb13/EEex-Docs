.. _EEex_GetSpellIcon:

===================================
EEex_GetSpellIcon 
===================================

::

   EEex_GetSpellIcon(resrefLocation)

Returns a string containing spell icon id

**Parameters**

* **resrefLocation** - resource reference of spell to return icon for

**Return**

Returns a string 

**Example**

::

   local icon = EEex_GetSpellIcon("SPWI304")
