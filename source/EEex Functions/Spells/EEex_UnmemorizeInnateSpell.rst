EEex_UnmemorizeInnateSpell 
===================================

::

   EEex_UnmemorizeInnateSpell(actorID, index)

Unmemorize an innate spell for an actor

**Parameters**

* **actorID** - id of actor that is unmemorizing spell
* **resref** - resource reference of spell to unmemorize

**Return**

None

**Example**

::

   EEex_UnmemorizeInnateSpell()
