===================================
EEex_IsSpellValid 
===================================

EEex_IsSpellValid()

Returns true if resource is a valid spell

**Parameters**

* **resrefLocation** - resource reference to check if a valid spell

**Return**

Returns true or false

**Example**

::

   local bSpellvalid = EEex_IsSpellValid("SPWI304") -- true
