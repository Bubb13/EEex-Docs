===================================
EEex_LearnClericSpell 
===================================

::

   EEex_LearnClericSpell(actorID, level, resref)

Adds a spell to known cleric spells for an actor

**Parameters**

* **actorID** - id of actor that is learning spell
* **level** - spell level
* **resref** - resource reference of spell to learn

**Return**

None

**Example**

::

   EEex_LearnClericSpell()
