===================================
EEex_MemorizeClericSpell 
===================================

::

   EEex_MemorizeClericSpell(actorID, level, resref)

Memorize a cleric spell for an actor

**Parameters**

* **actorID** - id of actor that is memorizing spell
* **level** - spell level
* **resref** - resource reference of spell to memorize

**Return**

None

**Example**

::

   EEex_MemorizeClericSpell()
