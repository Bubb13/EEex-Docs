.. _EEex_MessageBox:

===================================
EEex_MessageBox 
===================================

EEex_MessageBox(message)

Displays a message box to the user. Note: Suspends game until closed, which can be useful for debugging

**Parameters**

* **message** - text string of the message to output to user

**Return**

None

**Notes**

Windows OS only

**Example**

::

   EEex_MessageBox("Hello")
