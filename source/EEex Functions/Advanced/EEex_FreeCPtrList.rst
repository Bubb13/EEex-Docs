EEex_FreeCPtrList 
===================================

EEex_FreeCPtrList(CPtrList)

Frees the memory allocated by the given CPtrList pointer

**Parameters**

* **CPtrList** - pointer to the CPtrList to free

**Return**

None

**Example**

::

   EEex_FreeCPtrList()
