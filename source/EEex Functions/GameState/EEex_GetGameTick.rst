===================================
EEex_GetGameTick 
===================================

::

   EEex_GetGameTick()

Returns the game time tick (``m_gameTime``) value from ``m_pObjectGame``

**Parameters**

None

**Return**

Returns the game time tick value

**Example**

::

   local tick = EEex_GetGameTick()
