.. _EEex_GetGlobal:

===================================
EEex_GetGlobal 
===================================

::

   EEex_GetGlobal(globalName)

Gets a GLOBAL variable's value from the .GAM

**Parameters**

* **globalName** - the variable name to fetch the value for

**Return**

Returns the global variable

**Example**

::

   testervalue = EEex_GetGlobal("tester")
