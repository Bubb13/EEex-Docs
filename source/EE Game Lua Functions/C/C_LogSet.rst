.. _C_LogSet:

===================================
C\:LogSet 
===================================

Sets logging to a file instead of to console if logging is enabled
    
::

   C:LogSet(LogFile)


**Parameters**

* ``string`` *LogFile* - string containing log filename

**Notes**

See :ref:`C\:LogMessages<C_LogMessages>` to enable logging

**Example**

::

   C:LogSet("output.txt")

**See Also**

:ref:`C\:LogMessages<C_LogMessages>`
