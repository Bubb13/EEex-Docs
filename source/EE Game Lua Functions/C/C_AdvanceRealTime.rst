.. _C_AdvanceRealTime:

===================================
C\:AdvanceRealTime 
===================================

Advance time
    
::

   C:AdvanceRealTime(GameTime)


**Parameters**

* ``integer`` *GameTime* - amount of time in milliseconds to advance

**Notes**

Can be used to trigger banters and love talks etc

**Example**

Speed up 1 hour real time:

::

   C:AdvanceRealTime(60000)
