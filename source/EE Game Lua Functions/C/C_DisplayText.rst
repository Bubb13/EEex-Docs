.. _C_DisplayText:

===================================
C\:DisplayText 
===================================

Displays the specified text on screen
    
::

   C:DisplayText(text)


**Parameters**

* ``string`` *text* - string containing text to display

**Notes**



**Example**

::

   C:DisplayText("This is some text to display")
