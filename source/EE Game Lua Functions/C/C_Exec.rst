.. _C_Exec:

===================================
C\:Exec 
===================================

Execute console commands contained in the specified file
    
::

   C:Exec(File)


**Parameters**

* ``string`` *File* - string containing filename to execute

**Notes**



**Example**

Execute all the commands in the file ``test.txt``:

::

   C:Eval("test.txt")

**See Also**

:ref:`C\:Eval<C_Eval>`
