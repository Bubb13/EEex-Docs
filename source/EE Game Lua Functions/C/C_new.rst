.. _C_new:

===================================
C\:new 
===================================

Create a new instance of the CLUAConsole
    
::

   C:new()


**Parameters**

None

**Notes**

This is used in the game engine to assign ``C:`` to CLUAConsole:

::

   C = CLUAConsole:new()

Not recommended to use this

**Example**

::

   C:new()

:ref:`C\:CreateEngine<C_CreateEngine>`
