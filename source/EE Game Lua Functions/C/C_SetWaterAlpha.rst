C\:SetWaterAlpha 
===================================

Sets the alpha blend level for water transparency
    
::

   C:SetWaterAlpha(AlphaLevel)


**Parameters**

* ``integer`` *AlphaLevel* - value to set for alpha blending of water transparency

**Notes**

Not tested / verified

**Example**
