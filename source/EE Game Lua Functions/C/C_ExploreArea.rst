.. _C_ExploreArea:

===================================
C\:ExploreArea 
===================================

Reveals the entire map for the current area
    
::

   C:ExploreArea()


**Parameters**

None

**Notes**

This doesn’t remove the fog of war; if a creature is out of your field of vision it will still remain hidden

**Example**

::

   C:ExploreArea()

**See Also**

:ref:`C\:MoveToArea<C_MoveToArea>`
