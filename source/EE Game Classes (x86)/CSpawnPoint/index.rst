.. _CSpawnPoint_Overview:

Overview
---------------

.. note:: The **CSpawnPoint** classes handle x and y coordinates of spawn points, spawn point variables and spawn point arrays

The structures used for these class methods are :ref:`CSpawnPoint<CSpawnPoint>`, :ref:`CSpawnPointVar<CSpawnPointVar>` and :ref:`CSpawnPointArray<CSpawnPointArray>`
